# Review Assets

Review reusable assets for rot.

Trigger phrases:

- `review assets`
- `检查 asset rot`

Steps:

1. Read `workflows/review-assets.md`.
2. Check `indexes/asset_index.yaml` and `usage/asset-usage-log.yaml`.
3. Find assets with no usage, overlapping triggers, stale practice links, or missing adapter coverage.
4. Present recommended actions.
5. Do not deprecate, retire, merge, or broadly republish without approval.
