# Harvest Practices

Run the canonical practice harvesting workflow.

Steps:

1. Read `workflows/harvest-practices.md`.
2. Reconstruct the session.
3. Extract reusable candidates.
4. Search `indexes/practice_index.yaml`.
5. Decide discard, merge, create, supersede, or defer.
6. Update canonical files first.
7. Ask before promoting to `active`.
8. After approval, publish relevant adapters automatically.
9. Report candidates, decisions, changed files, and review needs.
