# Publish Practices

Publish adapters from current active practices.

Trigger phrases:

- `publish practices`
- `发布 practices`

Steps:

1. Read `workflows/publish-adapters.md`.
2. Include only `active` or `revised` practices.
3. Exclude `candidate`, `proposed`, `superseded`, and `archived` entries.
4. Update relevant adapters.
5. Report changed adapters and included canonical IDs.
