# Discover Assets

Discover repeated workflows that should become reusable assets.

Trigger phrases:

- `discover assets`
- `发现可打包资产`

Steps:

1. Read `workflows/discover-assets.md`.
2. Review available session/history evidence.
3. Search `indexes/asset_index.yaml` and `indexes/practice_index.yaml`.
4. Recommend the smallest suitable form: skill, subagent, automation, extend existing, skip, or defer.
5. Present a concise review list.
6. Wait for approval per candidate.
7. After approval, create or extend the asset, update asset index, and publish relevant adapters.
