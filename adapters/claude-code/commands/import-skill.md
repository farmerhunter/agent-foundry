# Import Skill

Evaluate an external skill, prompt pack, article, repository, or local folder.

Trigger phrases:

- `import skill <source>`
- `导入这个 skill <source>`

Steps:

1. Read `workflows/import-external-skills.md`.
2. Capture provenance and source.
3. Review license, security, and quality.
4. Extract useful candidate practices.
5. Deduplicate against `indexes/practice_index.yaml`.
6. Present a concise review list.
7. Wait for approval per candidate.
8. After approval, canonicalize, promote to `active` when applicable, update index, and publish relevant adapters.
